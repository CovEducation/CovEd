Note:
  After cloning, create a .env file with the MongoDB connection URI as follows:

  MONGO_URI : "$SomeConnectionStringHere"


# Starting the server

```
npm install
npm start 
```

To verify that the server is working correctly, in another terminal window:

```
npx webpack
npm run hotloader
```
